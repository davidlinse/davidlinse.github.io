Eclipse ADT Plugin Update

Sep 10th, 2010

If you are programming for the Android plattform in Eclipse and ADT you’ve probably seen various NullPointerExeptions when editing the AndroidManifest.xml or any other xml file from the res/ folder. To fix this go the “Help/Check for Udates” and install the latest version of the ADT-Plugin which was 0.9.8.v201008281755-53891 by the time of writing.
